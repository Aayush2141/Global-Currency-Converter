# 💱 Global Currency Converter

## 📌 Project Description
The Global Currency Converter is a web application that allows users to convert amounts between different currencies in real-time. It fetches live exchange rates from an external API and provides accurate and up-to-date conversion results through a simple and user-friendly interface.

---

## 🎯 Purpose
The purpose of this project is to demonstrate the use of APIs in a frontend application, along with handling asynchronous data, state management, and dynamic UI updates using React.

---

## 🔌 API Used
**ExchangeRate API**  
Endpoint: https://api.exchangerate-api.com/v4/latest/{BASE}

- Provides real-time exchange rates
- No authentication required (easy to use)
- Returns data in JSON format

---

## ✨ Features
- Convert between multiple currencies (USD, INR, EUR, etc.)
- Real-time exchange rate fetching
- User-friendly interface with dropdown selection
- Instant conversion results
- Swap currencies functionality (optional)
- Error handling for API failures

---

- React.js
